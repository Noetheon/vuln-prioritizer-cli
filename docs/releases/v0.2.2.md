# Release Notes: v0.2.2

## Focus

This patch release closes the remaining public-release hygiene gaps without changing prioritization behavior.

## Added

- `CODE_OF_CONDUCT.md`
- `.editorconfig`
- direct cache tests
- `py.typed` package marker

## Changed

- package metadata now includes project URLs, classifiers, and contributor-oriented authorship
- package licensing metadata now uses SPDX-style fields instead of deprecated setuptools patterns
- `make package` now builds both source and wheel artifacts
- `make release-check` now validates built artifacts with `twine check`

## Validation

- `make release-check`
- `python3 -m pre_commit run --all-files`

## Notes

- No scoring, filtering, or provider logic changed in this patch release.
